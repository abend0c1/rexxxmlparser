library 'Security' is
  program 'changePassword' is
    define data parameter
    1 userid   (A8)   In
    1 password   (A8)   In
    1 newPassword   (A8)   In
    1 result   ('SecurityContext')   Out
    end-define

  program 'verify' is /*Authenticate a userid
    define data parameter
    1 userid   (A8)   In
    1 password   (A8)   In
    1 result   ('SecurityContext')   Out
    end-define

  program 'authorise' is /*Authorise access to a resource
    define data parameter
    1 userid   (A8)   In
    1 password   (A8)   In
    1 resource   (A44)   In
    1 result   ('SecurityContext')   Out
    end-define

/*The result of a Security operation is a collection of return and reason codes

    1 userid   (A8)
    1 success   (L)
    1 returnCode   (B4)
    1 reasonCode   (B4)
    1 reason   (AV80)
    end-define
